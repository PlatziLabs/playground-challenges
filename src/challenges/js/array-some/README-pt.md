Neste desafio, você tem um array de números e deve retornar true se nos números do array existe pelo menos um número par.

A solução deveria ter os seguintes input e output:

Input:

```js
checkArray([1, 3, 5, 7, 10, 11]);
checkArray([1, 3, 5]);
checkArray([]);
```

Output

```js
true
false
false
```
