Tienes un array con palabras y tienes que retornar un array ordenado de la palabra con menos caracteres a la palabra con más caracteres.

La solución debería funcionar algo similar:

```js
solution([
  "Hola",
  "Viajar",
  "Sol",
  "Aprender"
]);
```

Output

```js
[
  "Sol",
  "Hola",
  "Viajar",
  "Aprender"
]
```
