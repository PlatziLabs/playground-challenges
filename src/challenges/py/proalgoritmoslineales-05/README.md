Dada una cadena `s` que contiene múltiples caracteres, determina si la apertura y cierre de paréntesis es válida.

Ejemplo 1:

```py
# Input
isValid("((a * c) + (b / 2))")

# Output
True
```

Ejemplo 2:

```py
# Input
isValid("(()")

# Output
False
```

Ejemplo 3:

```py
# Input
isValid(")(")

# Output
False
```
